Ultimate Program

This is a more complex program than the others.
It lists all available network adapters and shows the current internet statuses.

When an adapter is selected, it lists all available IP addresses and groups them by type. The host names of the IP addresses are automatically loaded and then saved. Shows adapter attributes, speed, name, ID, and description.

Another tab shows the packet traffic on the adapter. It displays the sender, receiver, connection type, and package content. Since there are many packets arriving per second, there is an optimizer that turns off the listing of packets when it is not visible, but it can be turned off. If a package arrives more than once, it merges them and indicates with a number that it has merged several packages. The traffic can also be viewed on a graph. It collects the sending and receiving IP addresses of the packets and adds them to the IP address list on the previous page, where we can manage them. If an IP address is selected, it is possible to ping the address and view the details, where you can query the location of the host, the more detailed name of the host, list the available ports, then connect to one and send or receive messages .
